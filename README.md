# 📌 Smart Attendance System

A QR-based Smart Attendance System that allows students to scan a QR code to mark attendance, while instructors can manage and export attendance records in real time.

---

## 🚀 Features

- 🔐 Login system (Student & Instructor)
- 📷 QR code generation (Instructor)
- 📱 QR scanning using phone camera (Student)
- ⛔ Duplicate attendance prevention
- ⏱ Session expiration system
- 📊 Live attendance display
- 📁 Export attendance as CSV
- 🗄 MySQL database integration

---

## 🛠 Technologies Used

- HTML, CSS, JavaScript
- Node.js & Express.js
- MySQL Database
- html5-qrcode (QR Scanner)
