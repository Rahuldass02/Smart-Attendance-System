# Smart Attendance System

This project is a QR-based attendance system.

## Features
- Login system (Student & Instructor)
- QR code generation
- Student scanning
- Duplicate prevention
- Live attendance display
- CSV export

## How to Run
1. Go to backend folder
2. Run: node server.js
3. Open login.html using Live Server

## Technologies
- HTML, CSS, JavaScript
- Node.js, Express
